PLDA and PLDA+

Introduction: PLDA is a parallel C++ implementation of Latent Dirichlet Allocation (LDA). We are expecting to present a highly optimized parallel implemention of the Gibbs sampling algorithm for the training/inference of LDA. The carefully designed architecture is expected to support extensions of this algorithm. We will release an enhanced parallel implementation of LDA, named as PLDA+, which can improve scalability of LDA by signiﬁcantly reducing the unparallelizable communication bottleneck and achieve good load balancing.
